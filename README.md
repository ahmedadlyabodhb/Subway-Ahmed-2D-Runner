
# Subway Ahmed - Pygame 🏃‍♂️💨

A 2D endless runner game inspired by the famous Subway Surfers, built entirely in Python using the **Pygame** library. The game features a clean design with neon colors and an integrated shop system.

## 🎮 Features
* **Dynamic Endless Gameplay:** The speed increases gradually as your score goes up, making the game more challenging over time.
* **Integrated Shop Menu:** Collect gold coins during your run to unlock new characters (Ahmed, Ninja, Fire, and Ghost).
* **Automatic Save System:** Your collected coins and high score are saved automatically to a local `save_pygame.json` file.
* **Performance Optimized:** Includes automatic memory cleanup for off-screen obstacles to ensure smooth 60 FPS gameplay on any device.

## 🕹️ Controls
* **A / Left Arrow:** Move to the left lane.
* **D / Right Arrow:** Move to the right lane.
* **W / Up Arrow:** Jump over low obstacles (bars/trains).
* **S / Down Arrow:** Slide under high obstacles (boxes).
* **M Key:** Open/Close the Shop Menu during gameplay.
* **R Key:** Restart the game after a Game Over.
* **ESC Key:** Exit the game safely from the Game Over screen.

## 🛠️ Requirements & Installation
To run this game locally, you need **Python 3** and the **Pygame** library installed on your machine.

1. Install Pygame via your Terminal or Command Prompt:
```bash
pip install pygame
```
